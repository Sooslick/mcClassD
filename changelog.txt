v1.2
- Адаптирован для совместимости с 1.18
- Добавлен параметр denyNetherTravelling. Если включен, жертва не сможет создавать новые порталы в нижнем мире
- Разделены стартовые инвентари: startInventory заменен на victimStartInventory и hunterStartInventory
- Добавлена настройка potionHandicap, которая позволяет настраивать эффекты стартового баффа жертвы и их длительность
- Добавлены публичные методы getVictimTeam и getHunterTeam для взаимодействия с таблицей очков в пользовательских режимах
- Исправлена ошибка, из-за которой игра начиналась после выхода всех игроков
- Механика подсветки игрока enableVictimGlowing теперь не использует эффект зелья
- Существенно оптимизирован алгоритм генерации стены для режима Escape The Wall
- Запрещен телепорт с помощью жемчуга края через барьер для режима Escape The Wall (MC-2164)
- Игра теперь завершается, если с сервера вышли все участвующие игроки. Теперь игру нельзя заблокировать
- Добавлена обработка исключений для пользовательских режимов. При возникновении исключения игра не будет зависать в "сломанном" состоянии
- Исправлена ошибка подбора спавнов. Теперь пустота не считается "безопасной" точкой для спавна
- Доработана очистка сущностей для выгруженных чанков
- Незначительные исправления внутриигровой логики
